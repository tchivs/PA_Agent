<threat_model>
## Trust Boundaries
| Boundary | Description |
|---|---|
| immutable façade DTO → presentation | panel 只能渲染，不能变更 risk/lifecycle/authority。 |
| user draft interaction → worker request | config actions必须按 active target/generation 安全应用。 |

## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-04-01 | Tampering | config/account stale render | high | mitigate | panel request callback carries generation/digest，render only matching immutable projection；workers e2e switch tests阻止旧 result。 |
| T-04-02 | Elevation of Privilege | widgets | high | mitigate | constructor only receives façade/request callback, no gateway/ledger/permit; summary and config have no submit controls。 |
| T-04-03 | Information Disclosure | labels/tables | high | mitigate | render safe DTO only，never raw errors/header/payload/credential; long text uses tooltip/copy after redaction。 |
| T-04-04 | Tampering | visual lifecycle/recovery | high | mitigate | stale/failed sections and persisted latch remain explicit；UI cannot call local READY/risk eligibility calculation。 |
| T-04-05 | Denial of Service | UI interactions | high | mitigate | click handlers only issue worker requests；busy/disabled/progress status, no blocking query, worker regression remains required。 |
</threat_model>
