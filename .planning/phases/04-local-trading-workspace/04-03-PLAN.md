<threat_model>
## Trust Boundaries
| Boundary | Description |
|---|---|
| background worker → Qt widget | 已排队结果可能比 target 选择和窗口生命周期更旧。 |
| durable projection → account display | 屏幕不能把只读事实改写为 eligibility/lifecycle 权威。 |

## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-04-01 | Tampering | stale callback | high | mitigate | e2e 测试覆盖 generation、target digest 和 UI-alive 防护，含 switch/close 后 success/error。 |
| T-04-02 | Elevation of Privilege | account controls | high | mitigate | account test断言跨产品 summary/read-only sections 没有 approval/submit 权威；未来 façade 不把 gateway 给 widget。 |
| T-04-03 | Information Disclosure | worker error rendering | high | mitigate | tests只允许 controlled safe error；实施由 SecretRedactor DTO 覆盖 raw exception。 |
| T-04-04 | Tampering | local recovery/terminal state | high | mitigate | account test要求 persisted latch、独立 reconciliation 和 cancellation non-terminal presentation。 |
| T-04-05 | Denial of Service | UI thread | high | mitigate | heartbeat、delayed fake、switch/close test 作为阻塞性安全 gate；任何 GUI-thread service 调用都使回归失败。 |
</threat_model>
