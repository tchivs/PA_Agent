<objective>
Define the exchange-neutral post-operation bridge before central projection wiring changes any lifecycle caller.

Purpose: the current gateway methods return bare `GatewayEvidence`, `SubmissionCoordinator.submit()` returns that evidence after its one call, `RecoveryService` separately looks up bare evidence, and no contract can carry a durable read-only projection reference. This plan inserts the generic result/reference seam while preserving the one-submit boundary; Plan 03-07 will attach the Paper batch-to-ledger projector to this seam.
Output: Generic operation result/reference and observer contracts, changed port/fakes/coordinator/recovery/cancellation/PaperGateway interfaces, and exact single-submit/read-only bridge regressions.
</objective>
