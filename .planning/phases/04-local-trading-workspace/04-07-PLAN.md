<threat_model>
## Trust Boundaries
| Boundary | Description |
|---|---|
| UI request → QThread worker | 主线程不能执行 blocking or authoritative trading work。 |
| worker error/result → UI slots | 延迟或敏感输出必须被绑定到仍有效的 workspace。 |

## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-04-01 | Tampering | stale result spoofing | high | mitigate | immutable generation+digest carry-through，panel-side UI-alive checks，switch/close e2e success/error tests为 blocking gate。 |
| T-04-02 | Elevation of Privilege | worker construction | high | mitigate | worker receives façade only，AppContext excludes gateway/ledger/permit exports to widgets；command façade retains coordinator chain。 |
| T-04-03 | Information Disclosure | exception signal | high | mitigate | SecretRedactor converts all exceptions to safe DTO before emit，tests assert safe controlled UI error。 |
| T-04-04 | Tampering | local recovery result | high | mitigate | worker only returns façade/persisted projection，does not synthesize terminal/latch state; stale result is discarded. |
| T-04-05 | Denial of Service | GUI event loop | high | mitigate | QThread execution, no UI handler wait, event-loop heartbeat and close/switch tests mandatory. |
</threat_model>
