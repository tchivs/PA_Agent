<threat_model>
## Trust Boundaries
| Boundary | Description |
|---|---|
| persisted settings/records → typed workspace contracts | 非秘密输入和 advisory 记录进入 future execution-adjacent UI 之前必须被验证。 |
| durable Paper/ledger truth → display projection | 只读事实不得变成风险或提交权威。 |

## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-04-01 | Tampering | future workspace callbacks | high | mitigate | 后续 worker 契约要求 generation、target digest、UI-alive 三重 guard；本阶段测试固定新旧 target 不可混用的 immutable DTO 前提。 |
| T-04-02 | Elevation of Privilege | projection/record reader | high | mitigate | 测试断言 reader 只能产生 typed snapshot，projection 没有 permit/lease/submit authority；后续 façade 只能走 consume → lease → coordinator。 |
| T-04-03 | Information Disclosure | settings/record error path | high | mitigate | 测试拒绝 secret/unknown settings，并要求 controlled ineligible reason；后续 worker 使用 SecretRedactor 安全 DTO。 |
| T-04-04 | Tampering | local lifecycle display | high | mitigate | projection 测试要求 persisted product/reconciliation/latch facts和独立 freshness，禁止本地 summary 推断 READY 或 terminal state。 |
| T-04-05 | Denial of Service | GUI-thread service path | high | mitigate | 后续 pytest-qt worker heartbeat/stale-result suite在 Wave 0 建立；实现必须没有 GUI-thread I/O。 |
</threat_model>
