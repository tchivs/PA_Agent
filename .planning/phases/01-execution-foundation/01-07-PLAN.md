## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| logical command caller to ledger admission | Caller input identifies logical work but cannot select its persistent remote identity. |
| canonical gateway evidence to projection | External evidence may update durable state only through lifecycle validation and cumulative Decimal projection rules. |
| protected authorization to abstract gateway | A future outward side effect is permitted only after an irreversible durable ledger transition. |
| typed account observation to SQLite | Only canonical balance/position data may become local durable evidence. |
