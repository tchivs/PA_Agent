<threat_model>
## Trust Boundaries
| Boundary | Description |
|----------|-------------|
| typed caller to validation service | A canonical command may enter only through OrderValidationService.validate(command); no public validation API permits the caller to provide or reuse venue metadata. |
| validation service to internal helper | Only the service passes the fresh RuleObservation.rules from its immediate lookup to the underscored helper for that invocation. |
| validation service to gateway | The service may request metadata but must never gain an order-submission capability. |
